# 任务目标
请基于上述来源材料，生成一张高质量图片，成果必须保存为{{expected_path}}。

# 执行流程
1. 先阅读和理解来源文件内容，提炼可视化主题、关键元素、风格和用途。
2. 根据来源内容组织出适合生成图片的提示信息。
3. 调用 {{IMAGE_TOOL}} 工具生成图片；如工具支持自定义文件名，请尽量让输出文件名与目标文件名保持一致。
4. 若工具实际输出路径与目标路径不一致，请在生成后将最终图片整理、移动或重命名到以下绝对路径：`{{expected_path}}`

# 图片生成要求
- 最终结果必须是图片文件，并与目标路径扩展名保持一致
- 优先保证主题明确、主体突出、构图完整
- 若来源材料不足，不要改为依据当前对话自由发挥；仅可在不偏离来源主题的前提下做最小必要补充
- 若来源中包含图片、页面、PDF 或文档，请优先提取其中最关键的视觉线索
- 不要输出与任务无关的闲聊，直接完成图片生成并确保最终文件位于目标路径
